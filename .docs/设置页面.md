# 设置页面

分类展示使用二级标题

设置页面需要的选项如下

## 界面设置

语言：
- 中文
- English

主题：
- 明亮主题
- 夜晚主题

记住上次窗口位置
- 开启
- 关闭

## 基本设置

扫描占用核心数
- Auto
- 1/2核心
- 所有核心

酒馆启动模式
`设置酒馆的启动方式`
- 正常模式
- 桌面程序
- 局域网服务
- 公网服务

## Git 设置

Git 环境信息卡片
- Git 版本
- Git 安装路径

Git 环境来源（如果识别系统有安装Git就默认使用系统Git，否则使用内置Git）
`可切换使用系统 Git 或内置 Git`
- 系统环境
- 内置环境（默认）
- 自定义环境

## NodeJs 设置

NodeJs 环境信息卡片
- NodeJs 版本
- NodeJs 安装路径

Node.js 环境来源（如果识别系统有安装NodeJs且版本大于v20就默认使用系统NodeJs，否则使用内置NodeJs）
`可切换使用系统 NodeJs 或内置 NodeJs`
- 系统环境
- 内置环境（默认）
- 自定义环境

NPM 环境信息卡片
- NPM 版本
- NPM 安装路径

NPM 源设置（可选）
- 官方源
- 淘宝源（默认）
- 腾讯源

## Github 设置

替换总开关
`开启后将在源地址前面加上加速地址，实现加速 Github 资源下载`
- 开启
- 关闭

替换节点列表（通过接口获取）（需要做缓存，缓存时间为3天）

接口：https://api.akams.cn/github
文档：https://api.akams.cn/docs/github.html

## 网络设置

代理设置
- 关闭
- 跟随系统
- 自定义代理

GitHub 连接测试
`不通过代理测试 GitHub 连接`
