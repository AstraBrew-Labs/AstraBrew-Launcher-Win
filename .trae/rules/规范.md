这是一个 rust + egui 项目
不要使用cargo run启动项目，也不要用于测试
只能使用cargo check检查项目
仅Windows平台，不用考虑其他平台。
 项目结构如下：
  data（根目录的版本）
   - lib 库目录
    - nodejs Node.js 目录
    - Git 目录 （内置MiniGit）
   - sillytavern SillyTavern 目录（存放酒馆核心文件）
   - logs 日志目录（存放日志文件）
   - settings.json 设置文件
  src
   - core 核心模块
    - ...
   - ui 自定义组件
    - ...
   - pages 页面
    - home.rs 首页
    - ...
   - utils 工具函数
    - ...
   - main.rs 主函数

页面要分文件写，不能全写在主入口里

要确保程序check通过

需要加上代码注释，注释要中文

图标库使用egui_phosphor

软件窗口界面，不能最大化
在16：9比例下，默认要1280x720，支持手动调整大小，最小不能超过800x600，最大不能超过1280x720
在4:3比例下，默认800x600，支持手动调整大小，最大不能超过1200x800
要适配高清屏，高分辨率屏，保持界面清晰度
要保证每个缩放都能正常显示
要适配多屏切换
要适配如果软件界面在副屏，但副屏断开，要自动切换到主屏显示

## 设置

要实现实时保存，用户在界面修改选项后，要自动保存到settings.json文件，然后界面中要有保存的提示，提示用户设置已保存。

配置参数模板如下：
```json
{
  "language": "zh_CN", // 语言，可选值：zh_CN, en_US
  "theme": "Dark", // 主题，可选值：Dark, Light
  "remember_window_pos": true, // 记住上次窗口位置
  "window_position": [0, 0], // 上次窗口位置，格式：[x, y]，默认屏幕中心
  "cpu_cores": "Auto", // 扫描占用核心数，可选值：Auto, Half, ALL
  "start_mode": "Normal", // 酒馆启动模式，可选值：Normal, Desktop, Local, Public
  "git_env": "System", // Git 环境来源，可选值：System, Builtin
  "nodejs_env": "System", // Node.js 环境来源，可选值：System, Builtin
  "npm_registry": "Taobao", // NPM 软件源
  "github_proxy": {
    "enabled": false, // 开关
    "url": "https://ghfast.top/" // 加速地址
  }, // 是否开启 GitHub 加速地址替换
  "proxy_type": "None", // 代理类型，可选值：None, Custom, System
  "custom_proxy": "" // 自定义代理地址, 格式：http(s)://ip:port
}
```
